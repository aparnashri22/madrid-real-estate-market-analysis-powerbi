# 🏠 Madrid Real Estate Market Analysis | Power BI Project

## 📌 Project Overview

This project analyzes the Madrid Real Estate Market using Power BI to identify pricing trends, evaluate the impact of property features, segment the housing market, and discover profitable investment opportunities.

The dashboard transforms raw housing data into interactive business intelligence, enabling investors, developers, and decision-makers to make data-driven decisions.

---

# 🎯 Business Objectives

The project answers the following business questions:

- How can duplicate and inconsistent property records be cleaned?
- How do luxury features such as swimming pools, gardens, and new developments affect property prices?
- Which house categories offer the best investment opportunities?
- How can an interactive dashboard improve real estate decision-making?

---

**Dataset:** Madrid Housing Dataset

## 2️⃣ Property Feature Price Analysis

Compared average property prices based on:

- Swimming Pool
- Garden
- New Development

### Key Findings

- Properties with pools average **€878.87K**
- Properties without pools average **€587.87K**
- Pool availability increases prices by nearly **50%**

- Properties with gardens average nearly **€1.9M**

- Newly developed properties average **€658.54K**

Pool availability was identified as the strongest pricing factor.

---

# 💼 Business Insights

✔ Premium properties with swimming pools command approximately **50% higher prices**

✔ Garden availability significantly increases property valuation

✔ New developments maintain premium market pricing

✔ Casa o Chalet provides the highest investment score

✔ Price per square meter helps compare property value more accurately than total price

---

# 💡 Business Recommendations

- Focus investment on Casa o Chalet and Duplex properties.
- Prioritize premium properties with pools and gardens.
- Consider new developments for long-term appreciation.
- Evaluate both property price and price per square meter before investing.
- Use interactive dashboards for continuous market monitoring.
